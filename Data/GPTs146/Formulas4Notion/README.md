# Formulas4Notion

Link: [Formulas4Notion](https://chat.openai.com/g/g-dOkgulsbb)

## Description

An agent to help write Notion database formulas.

## Features and Functions
* **Knowledge file**
  * `Notion Formula Docs.md` [14.21KB]

  * You can upload files to this GPT.
* **Browsing**: Enabling Web Browsing, which can access during your chat conversions.
