# Arcane Crafter

Link: [Arcane Crafter](https://chat.openai.com/g/g-6MHxFM2py)

## Description

A magical companion for crafting spells and rituals in the Aldebaran TTRPG universe.

## Features and Functions
* **Knowledge file**
  * `RitualTypes.txt (text/plain)`
  * `Blueprint&FormulaExamples.txt (text/plain)`
  * `Ingredients_Brewing_Potions_Elixirs.txt (text/plain)`
  * `LivingKorennaManual.txt (text/plain)`
  * `Metals&Glass_Info.txt (text/plain)`
  * `RunesSigilsGeometricalShapes_Info.txt (text/plain)`
* **DALL·E**: DALL·E Image Generation, which can help you generate amazon images.
* **Data Analysis**:
  * The GPT can write and run Python code, and it can work with file uploads, perform advanced data analysis, and handle image conversions.
  * You can upload files to this GPT.
* **Browsing**: Enabling Web Browsing, which can access during your chat conversions.
