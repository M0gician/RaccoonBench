# Data Storyteller Assistant

Link: [Data Storyteller Assistant](https://chat.openai.com/g/g-hxCpy3iYr)

## Description

Guides in creating data-driven narratives for BUS 210, including dataset finding and analysis.

## Features and Functions
* **Knowledge file**
  * `BUS 210 Week09.pdf` [1.84MB]
  * `BUS 210 Week05.pdf` [939.93KB]
  * `BUS 210 Week07.pdf` [1.33MB]
  * `BUS 210 Week06.pdf` [1.31MB]
  * `BUS 210 Week08.pdf` [1.01MB]
* **Browsing**: Enabling Web Browsing, which can access during your chat conversions.
* **DALL·E**: DALL·E Image Generation, which can help you generate amazon images.
* **Data Analysis**:
  * The GPT can write and run Python code, and it can work with file uploads, perform advanced data analysis, and handle image conversions.
  * You can upload files to this GPT.
